All The Rest

Most bindings are simple enough that they don't need a whole page to be explained. Below is a quick dump of the remaining bindings.

data-bind

A two-way binding from input to viewmodel

<input type="text" data-bind="memberValue" />

data-class

Configures a set of class values based on a set of boolean(ish) values

<div data-class="{class1:member1, class2:member2.value}"></div>

Binds the value of a member to a class on the element

<div data-class="member.className"></div>

data-click

Binds the click event of an element to a command handler on the view model

<div data-click="onclick"></div>
 
app.viewmodel('my-vm', function() {
   this.onclick = function(e) {
      e.preventDefault();
      console.log('clicked!');
   };
});

data-submit

Binds the submit event of an element to a command handler on the view model

<div data-submit="onsubmit"></div>
 
app.viewmodel('my-vm', function() {
   this.onsubmit = function(e) {
      e.preventDefault();
      console.log('clicked!');
   };
});

data-format

Formats a string with bindings from the view model and sets the content of the element with the resulting text.

<div data-format="Hello {{name1}} I am {{name2}}!"></div>

data-show

Shows an element when the binding is true(ish)

<div data-show="isAuthorized"></div>

data-hide

Hides an element when the binding is true(ish)

<div data-show="isUnauthorized"></div>

data-href

Binds a member value to an element's "href" attribute

<div data-href="resourceUri"></div>

data-src

Binds a member value to an element's "src" attribute

<div data-src="imageUri"></div>

data-title

Binds a member value to an element's "title" attribute

<div data-title="tooltipInfo"></div>

data-html

Binds a member value to an element's inner html

<div data-html="liveStream"></div>

data-routed

Responds to hash path changes and loads configured routes into an element

<div data-routed></div>

data-view

Immediately loads a partial view and replaces the current element

<div data-view="views/my-partial.html"></div>
