Getting Started

The first thing you need to do is create a new Application like so: 

window.myApp = new Application(document);

Now to create functionality you need to create a ViewModel in your javascript: 

window.myApp.viewmodel('getting-started', function() {
   this.myTitle = 'My First Application';
   this.updateTitle = function() {
      this.myTitle = 'My First Binding';
   };
});

Lastly we must wire up the new functionality to the DOM in your html 

<div view-model="getting-started">
   <h1 data-format="{{myTitle}}"></h1>
   <button data-click="updateTitle">Change The Words</button>
</div>
