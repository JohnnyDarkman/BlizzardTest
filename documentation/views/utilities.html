Utility Services utilities.js & html.js

These are general utility services that provide mostly shortcut, extension methods.

Utilities.js

utilities.nothing Always undefined
utilities.is(value, type) applies multiple methods of type matching
utilities.locked(target, member) returns true if the member can't be set on the target
utilities.readonly(target, member, func) creates a readonly getter property on the target
utilities.constant(target, member, value) creates a readonly value property on the target
utilities.proxy(target, source, member) proxies a value on the source to a matching member on the target
utilities.override(target, member, func) "intercepts" a function member on the target
utilities.observe(target, member, callback) creates an observable property on the target
utilities.path(source, path) follows a dot "." delimited member path starting from source
utilities.watch(target, path, callback) follows a path of observables and manages disposing callbacks
utilities.copy(obj) creates a shallow copy of an object
utilities.merge(from, to) performs a shallow merge from one object to another
utilities.toarray(items) attempts to convert any collection into an array
utilities.distinct(items) returns a filtered array of only "distinct" values
utilities.domdepth(node) returns how deep a node is in the document
utilities.domsort(nodes) returns a sorted array of nodes based on their depth in the document
utilities.domquery(node, selector) searches the node and its descendants (querySelectorAll)
utilities.indom(element) true if the element is "in" the document
utilities.observable(model) converts an existing object to an observable
utilities.surrogate(model) returns an observable proxy object

html.js

html.encode(str)HTML encodes a string
html.parse(str)parses a string into elements
html.getInputValue(element)returns the value of an input
html.setInputValue(element, value)sets the value of an input
html.setStyle(selector, style)adds a style rule to the document
html.removeClass(element, cls)removes a class from an element
html.addClass(element, cls)adds a class to an element
html.indom(element)true if the element is currently in the DOM
html.formatParameters(obj)Converts an object into a uri query string
html.formatUri(uri, params)Formats together a uri and query object/string
html.parseQuery(params)parses a uri query string into an object
