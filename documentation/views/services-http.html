services/http.js

A general AJAX wrapper that provides global and session request configurations. The service itself resolves per scope, but has a singleton registry which allows global configurations to be set for the entire application.

The HTTP service provides simplified REST methods: 

http.get   ('uri', {params}, function success(), [function error(), [{headers}, [{config}]]])
http.delete('uri', {params}, function success(), [function error(), [{headers}, [{config}]]])
http.post  ('uri', {body},   function success(), [function error(), [{headers}, [{config}]]])
http.patch ('uri', {body},   function success(), [function error(), [{headers}, [{config}]]])
http.put   ('uri', {body},   function success(), [function error(), [{headers}, [{config}]]])

The success and error methods are given a context (this) of an object containing all of the request/response data. 

{
   timeout: boolean,
   state: int,
   response: object,
   responseText: string,
   responseType: string,
   status: int,
   statusText: string,
   getHeaders: function
}
